## 🎯 Шаг 1: Создать канал (5 минут)

1. Открой Telegram → правый верхний угол «✏️» → **«Создать канал»**
2. **Название:** `GymFuel AI — Белок по фото`
3. **Описание** (из файла `03_tg_канал_описание.md`):
   ```
   AI-счётчик калорий для качков.
   Фотай еду — ИИ считает КБЖУ и спортпит.
   
   🎯 Для набора массы и сушки
   🥤 Знает твой спортпит
   🇷🇺 Русская кухня
   
   → Попробовать: https://gymfuel-ai.vercel.app
   ```
4. **Username:** `@gymfuel_ai` (или `@gymfuel_bot` если занят)
5. **Тип канала:** Публичный
6. **Аватар:** логотип с буквой G в оранжевом квадрате
