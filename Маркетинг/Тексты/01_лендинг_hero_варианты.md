# 01. Лендинг Hero — 3 варианта для A/B теста

**Проект:** GymFuel AI
**Секция:** Hero (экран 1, выше fold)
**Цель:** максимум кликов по primary CTA у качка 20-40 лет
**Tone:** прямой, мужской, без сиропа

---

## Общий eyebrow (над headline для всех вариантов)

```
🔥 Для качков, а не для худеющих тёток
```

---

## Вариант A — Провокационный

**Рекомендован как основной. Бьёт в боль, не даёт соскочить.**

### Headline
**Хватит гадать, сколько ты жрёшь белка.**

### Sub-headline
Думаешь, добрал 200г? По факту — 140. Фоткай тарелку — AI за 2 секунды покажет КБЖУ под твою цель. Плюс считает, сколько ложек протеина докинуть, чтобы закрыть норму.

### Primary CTA (на кнопке)
```
Начать считать белок →
```

### Secondary CTA
```
▶ Посмотреть, как работает
```

### Под кнопками (trust-row)
```
⭐ 4.8    10K+ качков    Бесплатно на старте    iOS · Android
```

### Микро-копирайтинг под CTA
```
Без карты. Без триал-ловушек. Просто скачай.
```

## Вариант B — Польза + цифры

**Для тех, кому провокация кажется перебором. Прямая выгода.**

**200г белка в день. Без таблиц, без калькулятора, без гугла.**

### Sub-headline
AI-счётчик для качков: фото тарелки → КБЖУ под цель (масса / сушка / поддержка). Знает твой спортпит и говорит, когда и сколько ложек сыпать в шейкер. Точность распознавания — 85-90%.

### Primary CTA (на кнопке)
```
Скачать бесплатно →
```

### Secondary CTA
```
▶ Как это работает (30 сек)
```

### Под кнопками
```
⭐ 4.8    10K+ качков    3.5M+ блюд в базе    iOS · Android
```

### Микро-копирайтинг под CTA


## Рекомендации по A/B тесту

| Вариант | Гипотеза | На какой трафик |
|---------|----------|-----------------|
| **A — Провокация** | Качки любят прямоту, CTR +20-30% | TG Ads, VK (тёплый), реклама в фитнес-каналах |
| **B — Цифры** | Конкретика снижает недоверие у опытных качков | Яндекс.Директ по запросам "калькулятор белка", "сколько белка" |
| **C — Прямолинейный** | Безопасный fallback, не отпугнёт холодного | Google/Яндекс по запросам "счётчик калорий", "приложение для массы" |

**Метрика победы:** клик по primary CTA / уникальный визит. Минимум 500 визитов на вариант, статзначимость 95%, срок теста — 14 дней.

**Сплит:** 33/33/34 равномерно. Если через неделю один вариант проседает на 40%+ — отключить раньше срока.
