## A. Лендинги фитнес/питание приложений

### Tier 1: Основные конкуренты

1. **MyFitnessPal** — https://www.myfitnesspal.com/
   - Hero с 3.5M 5-star отзывами, соц. доказательство (CNN, BBC), интеграция с 35+ приложениями
   - Палитра: White + dark text + gold stars
   - Структура: Hero → Social Proof → Features → Integrations → Results → FAQ

2. **YAZIO** — https://www.yazio.com/en
   - Минимализм, "Build healthy habits you'll love to keep"
   - Палитра: белый фон, нейтральная
   - Структура: Hero → Features → Success Stories → Download CTA

3. **Lifesum** — https://lifesum.com/ru/
   - Русский лендинг, 60M+ users, яркие фото еды
   - Палитра: Vibrant food photography + white space
   - Структура: Hero → Features → Social Proof → Testimonials → Media

4. **Cronometer** — https://cronometer.com/
   - "Science-backed nutrition tracking", 84 нутриента
   - Палитра: Dark navy (#262A3B) + light + green/blue accents
   - Структура: Hero → Feature Cards → Testimonials → Integrations → CTA

5. **Cal AI** — https://www.calai.app/
   - Dark mode, минимализм, 5M+ users, influencers
   - Палитра: Black/dark + white + bright CTA
   - Структура: Hero → Social Proof → Influencers → Features → Reviews

### AI-First калорийные счётчики

6. **Калория ИИ (CalorieAI.ru)** — https://calorieai.ru/

   - Структура: Hero → How It Works → Features → Comparison → Pricing → FAQ

7. **CalZen AI** — https://calzen.ai/en/
   - AI-first, счётчик по фото, 3M+ users, 4.8 rating
