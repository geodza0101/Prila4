## 2. Повторяющиеся паттерны у лидеров

### 2.1 Цветовые палитры (у 3+ конкурентов)

**Популярные комбинации:**
1. **White + Dark + Gold accents** (MyFitnessPal) — консервативно
2. **Light + Navy #262A3B** (Cronometer) — доверие, science-backed
3. **Light + Фиолет + Розовый** (Purrweb) — современно
4. **Dark + Bright CTA** (Cal AI) — AI-first, молодёжно

**Тренд:** От белого → navy + фиолет/розовый + dark mode

### 2.2 Типографика

✅ Все используют modern sans-serif:
- **Inter** (Purrweb, Cursor, Tailwind ecosystem)
- **SF Pro Display** (iOS-native)
- Иерархия: h1 (48-56px Bold) → h2 (32-40px) → h3 (20-24px) → body (14-16px)

### 2.3 Hero-секция

Стандартная формула:
1. Лого + минимальная навигация
2. Крупный заголовок (2-3 строки)
3. Primary CTA ("Download", "Start Free Trial")
4. Визуал: скриншот приложения ИЛИ фото еды
5. Часто: эмодзи, иконки

**CTA формулировки:**
- "Download the App" (Strong, Hevy)
- "Start Today" / "Start Free Trial" (MyFitnessPal)
- "CLAIM YOUR 3-DAY FREE TRIAL" (Cal AI)

### 2.4 Социальное доказательство

Все размещают:
- Количество юзеров ("5M+ users")
- Рейтинг ⭐ 4.8-4.9
- Логотипы СМИ (CNN, BBC, The Verge, CNET, Guardian)
- Цитаты инфлюенсеров (Cal AI) / публичных лиц (Strong)
- Testimonials карточки

**Размещение:** 2-3 секция после Hero

### 2.5 Структура лендинга (7-10 блоков)

1. Hero
2. Social Proof
3. How It Works / Features
4. Key Benefits
5. Testimonials / User Stories
6. Integrations

- Простые line-style иконки

## 3. Что ХВАЛЯТ / что РУГАЮТ

### 3.1 ХВАЛЯТ

**Дизайн и UX:**
- "Интерфейс приятный, как облегчённый VK + Telegram" (Cal AI)
- "Быстрая работа даже при слабом интернете"
- "Надежность и простота" (Strong, Reddit)
- "Интуитивный дизайн между подходами"
- "Точность отслеживания" (Cronometer)

**Социальность:**
- "Great for accountability and community" (Strava)
- Возможность видеть прогресс друзей (Hevy)

### 3.2 РУГАЮТ

**MyFitnessPal (боли):**
1. Неточные КБЖУ (пользовательская база, не проверена)
2. Агрессивный paywall (free tier ограничен)
3. Buggy в последнее время
4. Упражнения сложно логировать
5. Data breach 2018 (150M пользователей)

**Общие жалобы:**
1. Агрессивные подписки
2. Чрезмерная геймификация без результатов
3. Проблемы с приватностью (health data)
4. Перегруженный UI
5. Сложный onboarding

**Для калории-счётчиков:**
- Неточное распознавание фото
- Нет русской кухни (MFP)
- Платная стена на лучшие фишки

## 4. Специфика для качков (главное!)

### 4.1 Dark Mode — ОБЯЗАТЕЛЕН
- Зал часто тёмный
- Светлый экран во время тренировки бесит глаза
- Читаемость чёрного на белом плохая при ярком свете
- **Вывод:** Dark mode по умолчанию, light — опционально

### 4.2 Минимализм — КРИТИЧЕН
- Quality over quantity
- Логирование упражнения 15-30 сек
- Никакой геймификации (badges, challenges)
- Strong выигрывает у Hevy за простоту
- **Вывод:** Max минимализм, max функциональность

### 4.3 Жирные цифры — ГЛАВНОЕ
- Вес, повторения, подходы, калории, белок = главное
- Крупные ясные числа
- Графики важны, но вторичны
- Heat maps (Hevy) — супер популярная фишка
- **Вывод:** Typography выделяет метрики

### 4.4 НЕТ "розового"
- Lifesum женская/worden — мужики ненавидят
- Strong, Hevy — тёмные, мужские, нейтральные — winning
- **Вывод:** Палитра neutral-to-dark, без bright pink. Допускаются: navy, charcoal, dark purple

### 4.5 Быстрый ввод
- Кнопки 52px (Apple guidelines)
- Минимум текста
- Barcode scanner для спортпита
- Голосовой ввод желателен
- **Вывод:** UX под один палец (перчатки, потные руки)

### 4.6 Интеграции
- Strong/Hevy синхронизируются с Apple Health
- MyFitnessPal — с 35+ приложений
- Качки хотят единую экосистему
- **Вывод:** API-интеграции обязательны

### 4.7 Без психологии
- MyFitnessPal с motivational messages — не нравится
- Noom с психологией — не для качков
- **Вывод:** Tone = Direct, Honest, Data-driven. Можно Vibe-стиль (как Strong), но не мотивационный сироп

## 5. 🎯 Ключевые выводы для GymFuel (15 пунктов)

### 5.1 ЦВЕТОВАЯ ПАЛИТРА

```
Primary Background: #1A1A1A или #0F0F0F (тёмный нейтральный)
Secondary Background: #2A2A2A (контраст)
Primary Text: #FFFFFF (белый)
Secondary Text: #AAAAAA (серый для вторичного)

Accent Colors:
- Primary: #FF6B00 (оранжевый, энергичный) 
  ИЛИ #00D9FF (голубой, спортивный)
  ИЛИ #6BCB77 (зелёный, healthy)
- Secondary Accent: #FFB700 (золотой для highlights)

Proportion: 60% dark + 30% secondary + 10% accents
```

### 5.2 ТИПОГРАФИКА

```
Primary Font: Inter (Google Fonts)
Fallback: SF Pro Display (iOS)

Sizes:
- h1: 48-56px Bold
- h2: 32-40px Bold
- h3: 20-24px Semi-Bold
- Body: 14-16px Regular
- Metrics: 12-14px Semi-Bold (ВАЖНО!)

Weights:
- Regular (400)
- Semi-Bold (600) — метрики
- Bold (700) — заголовки
- Extra-Bold (800) — КАЛОРИИ, БЕЛОК (главные числа)
```

### 5.3 TONE OF VOICE

❌ НЕ: "Давай тренироваться с улыбкой!", "Поздравляем с нормой!"
✅ ДА: "Белок: 145/200г", "Не добрал 40г белка. 1 ложка протеина. Пей."

Стиль: Direct, Honest, Data-driven, No BS. Опционально — Vibe-провокация (жёсткий тренер).

### 5.4 UX-ПАТТЕРНЫ

**1. Логирование упражнения: 15-30 сек**
- Экран 1: Упражнение (autocomplete)
- Экран 2: Вес/подходы/повторения
- Финал: "✓ Logged"

**2. Дневник еды: 10-20 сек**
- Кнопка "📷 Фото" → распознавание → подтверждение

**3. Главный экран: только метрики**
```
Калории: 1800 / 2000
Белок: 145 / 200г
Углеводы: 220 / 250
Жиры: 60 / 70
[+ ЕДА] [+ ТРЕНИРОВКА]
```

**4. Графики: во вторичные экраны**
- Главный = только цифры
- Свайп → неделя/месяц

**5. Спортпит-калькулятор в дневнике**
- "Протеин: осталось 40г" → [Выбрать напиток]
- УНИКАЛЬНАЯ ФИШКА GymFuel

**6. Dark Mode по умолчанию**

### 5.5 АНТИПАТТЕРНЫ (чего избегать)

🚫 Слишком много экранов (max 5 tab'ов)
🚫 Навязчивые пуши ("Выпейте воды!")
🚫 Длинный onboarding (max 3 шага)
🚫 Сложные формы (max 3 поля)
🚫 Bright colors (неон, pink)
🚫 Много текста (max 20 слов)
🚫 Скрытые фичи в free tier
🚫 Реклама в free
🚫 Gamification без связи с результатами
🚫 Интеграция с соцсетями (не хотят делиться весом в VK)

### 5.6 ОБЯЗАТЕЛЬНО в MVP

**Дизайн:**
- Dark mode по умолчанию (#1A1A1A)
- Белый текст, высокий контраст
- Оранжевый или голубой accent
- Inter / SF Pro Display
- h1 48-56px, Body 14-16px, Metrics BOLD
- Минимум украшений
- Кнопки 52px
- 5 экранов: Home, Log Food, Log Workout, Stats, Profile
- Apple Health интеграция
- Barcode для спортпита

**Функциональность:**
- Упражнение 15-30 сек
- Еда 10-20 сек
- 4 метрики на главном (Ккал, Белок, Углеводы, Жиры)
- Недельный/месячный график
- Спортпит-калькулятор в дневнике

## 8. ИТОГО

**GymFuel должен быть:**
1. **Dark-first** — оптимизирован под зал
2. **Минимальный** — как Strong + уникальная фишка спортпита
3. **Честный** — tone of voice прямой, без BS
4. **Для качков** — без розового, без геймификации
5. **Русский** — локализация как Калория ИИ

**Главное УТП в дизайне:**
Никто не объединяет **еда + спортпит + упражнения** в одном интерфейсе. Strong = тренировки, MyFitnessPal = еда. GymFuel = ВСЁ в тёмной лаконичной системе.

**Риск:** Перегруз. Решение: спортпит-часть максимально простая (бренд → кол-во → готово).
